1. Which terminal client did you choose? Why?

I chose Windows Terminal with PowerShell because it is modern, fast, and works well on Windows. It allows me to manage files, run Git commands, and work on projects efficiently in one place.

2. What customizations (if any) did you make?

I made a few simple customizations to improve readability:

Theme: I set the theme to One Half Dark, which makes the terminal easier on the eyes
Font size: I increased the font size once to make text more readable

I did not change the default font, background, or cursor style.

3. Aliases or plugins

I did not set up any aliases or plugins yet. I focused on learning the basic terminal commands first before adding more advanced configurations.

4. What was the most useful command you learned today?

The most useful commands I learned today are:

-git status – it shows which files have been modified, staged, or are untracked
-cd – to navigate between folders
-git add . – to stage changes
-git commit -m "message" – to save my work

While working on my reflection task, I edited some files in my project folder. I used cd to navigate to the correct directory, then ran git status to check which files were changed. I noticed which files were modified and confirmed everything before committing. Then I used: git add ., git commit -m "updated terminal reflection".

This helped me make sure I only committed the correct changes and kept my work organized.
